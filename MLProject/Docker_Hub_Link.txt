# Docker Hub Repository Link

Repository: https://hub.docker.com/r/[YOUR_USERNAME]/penguin-ml-project

Image Name: penguin-ml-project:latest

## Usage:

### Pull Image:
```bash
docker pull [YOUR_USERNAME]/penguin-ml-project:latest
```

### Run Container:
```bash
docker run -p 5000:5000 [YOUR_USERNAME]/penguin-ml-project:latest
```

### Run with MLflow UI:
```bash
docker run -p 5000:5000 [YOUR_USERNAME]/penguin-ml-project:latest conda run --no-capture-output -n penguins-ml-env mlflow ui --host 0.0.0.0
```

## Build Instructions:

### Build locally:
```bash
docker build -t penguin-ml-project:latest .
```

### Tag for Docker Hub:
```bash
docker tag penguin-ml-project:latest [YOUR_USERNAME]/penguin-ml-project:latest
```

### Push to Docker Hub:
```bash
docker push [YOUR_USERNAME]/penguin-ml-project:latest
```

Replace [YOUR_USERNAME] with your actual Docker Hub username.
